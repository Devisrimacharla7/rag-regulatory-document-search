# RAG Regulatory Document Search

An AI-powered Retrieval-Augmented Generation (RAG) application that enables users to search regulatory documents using semantic search and Large Language Models.

## Project Overview

This project demonstrates how Retrieval-Augmented Generation (RAG) can be used to answer questions from large collections of compliance and regulatory documents.

## Tech Stack

- Python
- FastAPI
- LangChain
- OpenAI
- FAISS

## Planned Features

- Document Upload
- PDF Processing
- Embedding Generation
- Vector Search
- Question Answering
- Chat Interface
